# Clinical Decision Support System (Graduation Project)

## Overview
Graduation project focused on developing an intelligent decision support system for plastic surgery clinics.

The project covers full lifecycle of system analysis: from business process modeling to prototype implementation.

## Business Analysis Scope
- AS-IS / TO-BE process modeling (BPMN)
- Requirements elicitation and documentation
- Functional and non-functional requirements
- Stakeholder and process analysis

## System Design
- UML diagrams (use case, class, interaction)
- Database design (ER-model, PostgreSQL)
- Business rules and logic design

## Data & Prototype
- Python-based prototype
- Synthetic dataset generation using scikit-learn
- Rule-based decision logic

## Project Evaluation
- Economic efficiency analysis (ROI, NPV)
- Project planning (Gantt chart)

## Repository Structure
- /bpmn — process models
- /uml — UML diagrams
- /database — ER model and schema
- /requirements — system requirements
- /python — prototype implementation
- /report — thesis documentation
