# Clinical Decision Support System (Graduation Project)

## Project Description
This project is a graduation thesis focused on the development of an intelligent decision support system for plastic surgery.

The system is designed to support medical decision-making through business process analysis, data modeling, and rule-based logic.

## Goals
- Analyze business processes in a plastic surgery clinic
- Design AS-IS and TO-BE processes (BPMN)
- Define system requirements
- Design database structure (ER-diagram)
- Develop a prototype in Python
- Build a knowledge base using production rules
- Perform economic evaluation (ROI, NPV)

## Technologies
- BPMN
- UML
- SQL / PostgreSQL
- Python
- scikit-learn
- Draw.io / Miro
